lingdang
========

狗铃订餐  

iOS 平台开发进行中，欢迎加入！  
另，求  web or Android 平台有人认领~

### 原则
- 只使用原生 UI 控件和第三方库，不定制复杂控件
- 功能不求最好，只求最简



### 数据
- 服务器端使用第三方平台 [Parse.com](https://www.parse.com/)
- 目前有四张表 user，food， shop， order

### iOS 开发上手
- 代码拉到本地后缺少 BXConfig.h 文件（存有 server keys），联系 zengming 把文件放到 ios/lingdang 目录下即可。
